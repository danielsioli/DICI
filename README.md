# pyDICI
Pacote para enviar dados ao DICI

## Install 

```
pip install pyDICI
```

## Use
Importar pacote
```
import pyDICI as dici
```
Enviar Arquivo
```
argumentos = {'arquivos':'C:\\arquivos.xlsx','hm':True}
dici.enviar_dados(argumentos)
```
Olhar o Help
```
dici.help()
```
